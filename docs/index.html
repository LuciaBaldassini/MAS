Description of the riddle

You and other individuals have been captured by super-intelligent aliens! The aliens think agents look quite tasty, but luckily for you, their civilization forbids eating highly logical and cooperative beings. They are not sure whether you qualify, so they decide to give you all a test to figure that out. The aliens tell you the following:

Each of you will have either a blue or a red hat, assigned randomly and no one will be told how many of each colour there are. When I tell you to begin, each of you must guess the colour of your hat, starting from the person in the back and moving down the line. Donât even think to say words other than blue or red or signal something with intonation for instance.
Else, you will all be eaten immediately!
If at least the total number of you minus 1 guess the colour correctly, you will be spared. You are given a couple of minutes to discuss a strategy before we start!â

Strategy

The strategy to win this game and be freed by the aliens is to agree that whichever individual has to give an answer, says:
  • Blue to indicate that he sees an odd number of red hats in front of him.
  • Red to indicate that he sees an even number of red hats in front of him.

As all agents are trustworthy, each agent can fully rely on the fact that each agent applies the strategy accordingly (and thus, honestly).

Since the tallest agents does not hear any information from another agent before he has to speak, he cannot be entirely sure about his hat colour. Therefore, the first agent to answer (i.e.Â the tallest) has a large chance of guessing his hat color incorrectly. However, this is not a big problem, since the aliens allowed the agents to make one mistake as a group in total. All other agents can say with full certainty which colour their hat is. Hence, there will not be made more than 1 mistake by the group. Namely, on the basis of hearing the previous answer(s) and seeing the hats in front of him, each agent can update his private knowledge and determine his own hat colour.

The gameâs goal is to have as many of the agents correctly guess their hat colour. The game is successfully played if the number of hats minus 1, is collectively reached. Therefore, using the technique mentioned above, the game is always played successfully, since only the tallest agent is unsure.

Implementation

To solve the riddle we built a system that is run from the command line. At the beginning, the system will ask the user whether (s)he would like to read the descirption of the riddle or not. Next, the user can select the amount of agents and whether or not to assign a hat colour for each agent. In case the user does not want to manually input a hat colour, these will be assigned randomly. The system will then show a table containing each agent, their hat colour and how many red and blue hats each agent see. Throughout the programme, we adopt the convention that the tallest agent is represented by the agent with the highest number and the shortest agent is represented by the agent with the lowest number.

Each agent is created from a class Agent() which stored the agent number, the colour of the hat and the number of hats in front of him. The main programme contains the dialogues with the user and a several functions. The function assignRandomHat and assignUserHat assign a random hat colour or the colour input by the user, respectively. The function showHatDistribution can be called to print the distiributions of the hats among the agents. Lastly, the function countHat counts, for each agent, the amount of blue and red hats seen by the agents.
